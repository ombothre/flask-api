SVM API

This Flask API is designed for Delta Corp to predict whether a new user will buy their items or not based on their gender, age, and salary.

To make a prediction, send a POST request to the route /predict with the following JSON body:

{
  "input": {
    "gender": "Male",
    "age": 32,
    "salary": 40000
  }
}

You can use the api-test.py script from this repository or utilize services like Postman to test the API.
